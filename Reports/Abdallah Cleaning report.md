README - Cleaned Datasets

Files Included:
- CUSTOMER.csv: Cleaned signup dates and normalized locations.
- STORE.csv: Standardized store types and cities.
- PROMOTION.csv: Fixed date ranges and discount values.
- SUPPORT_TICKET.csv: Added resolution time in days.
- EXTERNAL_FACTOR.csv: Handled missing weather and economic indices.
- MARKETING_CHANNEL.csv: Cleaned channel names and cost tiers.

Usage:
These files are ready for direct import into Power BI, Tableau, or SQL databases.
